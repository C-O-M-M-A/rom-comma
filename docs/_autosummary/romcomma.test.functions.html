romcomma.test.functions§

Contains a suite of functions designed to test_data GSA and order reduction with high-dimensional distributions.

All functions herein are taken from https://salib.readthedocs.io/en/latest/api/SALib.test_functions.html.

Scalar(call, loc, scale, M_, **kwargs)

A scalar test function, which calls self.call(self.loc + self.scale * X[:, :M], **(self.kwargs | kwargs).

Vector(name, **kwargs)

A vector test functon, which is little more than a named dictionary of Scalar functions, such that Vector(X, **kwargs) concatenates Scalar(X, **kwargs) for each dictionary item key: Scalar.
